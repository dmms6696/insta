# Student Card News Images

Put student-made card news image files in this folder.

Recommended file names:

```text
p001-01.png
p001-02.png
p001-03.png
p002-01.jpg
```

Then enter the relative path in the Google Sheet `Cards` tab:

```text
cards/p001-01.png
```

Use `imageAlt` for a short description of the image.

Supported formats in GitHub Pages: `.png`, `.jpg`, `.jpeg`, `.webp`, `.gif`.
